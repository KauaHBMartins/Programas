O que é Internet e WEB? São as mesmas coisas? Explique de forma clara e evidencie as diferenças.

R: A internet é uma rede mundial em que liga todos os comupatadores em uma rede. Já a Web é uma "teia" com páginas contendo texto, imagens, figuras e assim vai. Em sumo a Web são informações.

O que é arquitetura cliente-servidor utilizada na web ? Explique o carregamento de uma página passo a passo.

R: É uma arquitetura de uma aplicação distribuída. Sendo assim, na rede possuem forneceredores de recursos ou serviços a rede que eles se chamam servidores. E existem quem requere os serviços que são os clientes.

O que é um tag HTML?

R: A tag HTML são intruções que são mandadas para o navegador para a formatação do texto. Como por exemplo a definição da tag que existem várias que são usadas para indicar a estrutura do site.

O que é HTML e qual a estrutura básica de um documento HTML?

R: As principais tags são : "html" , "head" e "body" e a instrução "! DOCTYPE".

Discuta com colega o que são tags semânticas do HTML.

R: As tags semânticas são o uso correto das tegs em HTML para indicar o significado do conteúdo de uma forma clara e correta.

Explique como é criado um link no html sobre um parágrafo.

R: É criado link em html sobre um parágrafo usando a tag "a" e já sendo atribuida a tag "href" que é conhecida como "HyperText Reference" ou "Target" que conterá o endereço web apontado.
